# Heroe vs Cacas Malvadas

Un minijuego en HTML5/Canvas donde un héroe derrota cacas malvadas para sumar puntos. En el nivel 5 aparece la **Caca Boss Final**, una caca mega fuerte que dispara y requiere muchos impactos para caer.

## Cómo jugar (versión súper fácil)
- Haz doble clic en `index.html`. Se abrirá el juego en el navegador como si fuera una imagen.
- El héroe es el cuadrito de color: **flechas** o **WASD** lo mueven, **ESPACIO** hace "piu piu".
- No dejes que las cacas toquen al héroe. Si las esquivas y les pegas, ganas puntos.
- Cada nivel tiene más cacas. En el nivel 5 aparece la **Caca Boss Final** (una caca gigante muy fuerte).
- Si pierdes o ganas, pulsa **R** o **ENTER** para empezar otra vez desde el principio.
