# Heroe vs Cacas Malvadas

Un minijuego en HTML5/Canvas donde un héroe derrota cacas malvadas para sumar puntos. En el nivel 5 aparece la **Caca Boss Final**, una caca mega fuerte que dispara y requiere muchos impactos para caer.

## Cómo jugar
- Abre `index.html` en tu navegador favorito.
- Mueve al héroe con las flechas o **WASD**.
- Dispara con **ESPACIO**.
- Derrota a todas las cacas para subir de nivel; evita sus disparos.
- Sobrevive hasta el nivel 5 y vence a la **Caca Boss Final** para ganar.
- Si pierdes o ganas, pulsa **R** o **ENTER** para reiniciar la partida.
